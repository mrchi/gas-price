# 油价展示 Web 应用设计文档
## 项目概述
本应用是一个纯前端的油价展示工具，展示国内成品油价格和国际布伦特原油价格的变化趋势，帮助用户了解油价走势和两者的相关性。
- 部署目标：GitHub Pages
- 技术栈：HTML + Tailwind CSS + Chart.js
- 数据更新方式：通过 GitHub Actions 定期自动更新 JSON 数据文件
## 功能列表
### 已实现功能
1. ✅ 北京地区当前 92/95/98 号汽油价格展示
2. ✅ 过去一年国内成品油价格历史趋势折线图
3. ✅ 涨跌幅展示（红色上涨、绿色下跌、灰色持平）
4. ✅ 支持切换显示不同标号的成品油价格曲线
### 新增功能
1. ✅ 布伦特原油当前价格展示
2. ✅ 布伦特原油历史价格趋势展示（与成品油同图对比）
3. ✅ 双 Y 轴支持：左侧 Y 轴为成品油价格（元/升），右侧 Y 轴为原油价格（美元/桶）
4. ✅ 原油价格涨跌幅自动计算
5. ✅ 原油价格显示/隐藏开关：支持独立控制布伦特原油价格曲线的显示和隐藏
## 数据结构设计
### 1. china_gas.json（成品油价格数据文件）
```json
{
  "province": "北京",
  "price_history": [
    {
      "update_time": "2026-03-24 00:00:00",
      "gas_92": { "price": 8.57, "change": 0.93 },
      "gas_95": { "price": 9.12, "change": 0.99 },
      "gas_98": { "price": 10.62, "change": 0.99 }
    }
    // 其他历史数据...
  ]
}
```
### 2. crude_oil.json（新增文件）
```json
{
  "type": "布伦特原油",
  "unit": "美元/桶",
  "price_history": [
    {
      "update_time": "2026-03-24 00:00:00",
      "price": 85.6
    }
    // 日期与 gas_price.json 中的调价日期一一对应
  ]
}
```
## UI 设计
### 当前价格区域
- 在现有 3 个成品油价格卡片右侧新增第 4 个原油价格卡片
- 卡片内容：原油类型、当前价格、涨跌幅（绝对值 + 百分比）
- 颜色规则与成品油保持一致
### 图表区域
- 保留左侧 Y 轴：单位`元/升`，展示成品油价格
- 新增右侧 Y 轴：单位`美元/桶`，展示布伦特原油价格（橙色系）
- 图例新增"布伦特原油"选项，支持显示/隐藏
- 悬浮提示框同时展示对应日期的所有油品价格信息
## 交互逻辑设计
1. **数据加载**：页面加载时并行请求两个 JSON 文件，按`update_time`匹配合并
2. **涨跌幅计算**：原油涨跌幅在前端实时计算，与上一期价格对比
3. **图表交互**：支持独立控制每种油品（包括原油）的显示/隐藏
4. **更新逻辑**：两个数据文件独立更新，通过 GitHub Action 分别维护，无需修改前端代码
## 后续维护说明
- 成品油价格数据源：国家发改委调价公告
- 原油价格数据源：国际原油交易市场公开价格
- 两个文件的日期必须一一对应，保证图表展示准确
