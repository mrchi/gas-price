# 油价展示Web应用

一个纯前端的油价展示工具，展示国内成品油价格和国际布伦特原油价格的变化趋势，帮助用户了解油价走势和两者的相关性。

## 功能特性

✅ **当前价格展示**
- 北京地区92/95/98号汽油实时价格
- 布伦特原油实时价格（美元/桶）
- 自动计算涨跌幅，红色上涨、绿色下跌、灰色持平

✅ **历史趋势图表**
- 过去一年成品油价格变化折线图
- 布伦特原油价格变化折线图，与成品油同图对比
- 双Y轴设计：左侧显示成品油价格（元/升），右侧显示原油价格（美元/桶）
- 支持独立切换显示/隐藏任意油品的价格曲线
- 鼠标悬浮显示详细数据：价格、涨跌值、涨跌幅

## 技术栈

- 纯前端实现，无后端依赖
- HTML + Tailwind CSS v3
- Chart.js v4.4.8 图表库
- 数据源：静态JSON文件，通过GitHub Actions定期自动更新

## 本地运行

```bash
# 启动本地开发服务器
python3 -m http.server 8000

# 访问 http://localhost:8000
```

## 数据更新

- 成品油价格数据源：国家发改委调价公告
- 原油价格数据源：国际原油交易市场公开价格
- 通过GitHub Actions每日自动更新数据文件
